# Sales Analytics Project

## Overview
This project analyzes sales performance using Python, SQL, and Power BI to generate actionable business insights.

## Tools & Technologies
- Python (Pandas, Matplotlib)
- SQL (SQLite)
- Power BI

## Key Features
- Data cleaning and preprocessing using Python
- SQL-based analysis for sales, profit, and orders
- Interactive Power BI dashboard with KPI cards
- Monthly and regional sales trend analysis

## Dashboard KPIs
- Total Sales
- Total Profit
- Total Orders

## Outcome
This project demonstrates end-to-end data analysis skills, from raw data to business-ready dashboards.
